# Security Awareness Program Design

## Overview

This repository contains my Week 5 internship assignment that I completed during the Sohail Smart Solutions Summer Training Program 2026.

The project tends to mainly focuses on designing a six-month Security Awareness Program for EduTrack Learning Solutions, which, as I mentioned, is a fictional mid-sized education technology company. The program is able to emphasize the human aspect of cybersecurity by combining the security awareness training and phishing simulations, along with the performance metrics and continuous improvement activities.

The awareness program aligns with ISO/IEC 27001:2022 Annex A.6.3, as well as supports the organization's Information Security Management System (ISMS).

---

## Repository Contents

- **security-awareness-program.md** – Summary of the Security Awareness Program.
- **monthly-awareness-plan.csv** – Six-month awareness schedule.

---

## Program Objectives

- Improve employee cybersecurity awareness.
- Reduce phishing and social engineering risks.
- Promote secure handling of sensitive information.
- Encourage timely reporting of suspicious activities.
- Strengthen compliance with ISO/IEC 27001.
- Build a long-term security-aware culture.
