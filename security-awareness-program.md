# Security Awareness Program

## Introduction

EduTrack Learning Solutions is committed to strengthening its overall security posture by being able to address the human aspect of cybersecurity. This six-month Security Awareness Program has been developed to be able to improve the employee awareness of common cyber threats while also supporting the organization's Information Security Management System (ISMS). The program corresponds to the ISO/IEC 27001:2022 Annex A.6.3, as well as also promotes a culture in which every employee is able to understand their role in protecting the organizational information.

---

## Program Objectives

- Improve employee understanding of the cybersecurity risks.
- Reduce phishing, as well as social engineering incidents.
- Encourage secure handling of any sensitive information.
- Promote timely reporting of any suspicious activities.
- Strengthen compliance with corporate security policies.
- Support continual improvement of security awareness.

---

## Target Audiences

- All Employees
- Managers and Team Leaders
- IT and Security Teams
- Human Resources
- Finance Department
- New Employees

---

## Six-Month Awareness Plan

| Month | Topic |
|--------|-------------------------------|
| Month 1 | Information Security Fundamentals |
| Month 2 | Phishing and Social Engineering |
| Month 3 | Password Security and Multi-Factor Authentication |
| Month 4 | Data Classification and Secure Information Handling |
| Month 5 | Physical Security and Remote Working |
| Month 6 | Incident Reporting and Security Culture |

---

## Phishing Simulation

A simulated phishing campaign is conducted during the second month of the program. The employees receive a realistic email appearing to come from the Human Resources department, which basically is requesting them to review an updated 'work from home' policy. The exercise is able to evaluate whether the employees can successfully identify suspicious emails, as well as avoid clicking malicious links, and even report the message through the organization's reporting process.

Employees who are able to successfully identify and report the phishing email are able to reinforce good security practices, while those who interact with the message tend to receive additional awareness training to be able to improve future performance.

---

## Security Culture Metrics

The organization is able to measure the effectiveness of the program using:

- Training completion rate
- Phishing click rate
- Phishing reporting rate
- Assessment quiz scores
- Security incident reporting rate
- Policy acknowledgement rate
- Repeat phishing failures

These metrics are able to help the management evaluate any improvements in employee awareness, as well as identify areas requiring additional support.

---

## Continuous Improvement

Security awareness is an ongoing process rather than just a one-time activity. Feedback from employees, along with phishing simulations and audit findings, as well as security incidents, is reviewed regularly to be able to improve the future awareness sessions. Training materials are updated whenever there are new cyber threats emerge to be able to ensure that the employees remain prepared for evolving security risks.

---

## Conclusion

An effective Security Awareness Program is able to complement the technical security controls by being able to reduce the risks that are associated with human error. Through regular training and phishing simulations, along with the measurable performance indicators, as well as continuous improvement, EduTrack Learning Solutions is able to strengthen its overall security culture while also supporting the compliance with ISO/IEC 27001:2022, as well as the organization's Information Security Management System.
